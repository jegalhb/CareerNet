# CareerNet 프로젝트 소개

## 프로젝트 개요

CareerNet은 진로 탐색이 막막한 중고등학생, 대학생, 취업 준비생이 자신의 흥미 유형을 검사하고, 추천 직업과 실제 현직자 멘토링까지 이어갈 수 있도록 만든 진로 탐색 플랫폼입니다.

단순히 직업 정보를 나열하는 것이 아니라, 사용자의 검사 결과를 저장하고, 직업 추천, 직업 상세 정보, 관련 멘토, 채용공고, 커뮤니티, 개인 로드맵까지 하나의 흐름으로 연결하는 것을 목표로 합니다.

## 핵심 문제

- 학생이 자신의 흥미와 강점을 직업 선택으로 연결하기 어렵습니다.
- 직업 정보는 많지만 실제 준비 순서와 현직자 경험을 함께 확인하기 어렵습니다.
- 검사 결과, 관심 직업, 멘토링 신청 내역이 흩어져 있으면 진로 준비가 지속되기 어렵습니다.

## 해결 방향

- Holland 진로 검사 결과를 기반으로 직업을 추천합니다.
- 직업 상세 화면에서 필요한 역량, 전공, 로드맵, 멘토 정보를 함께 제공합니다.
- 현직자 멘토링 신청 흐름을 제공합니다.
- 마이페이지에서 검사 결과, 관심 직업, 멘토링 내역, 진로 로드맵을 관리합니다.
- 채용공고와 커뮤니티를 통해 실제 시장 정보와 사용자 경험을 함께 제공합니다.

## 주요 사용자

- 진로 방향을 정하고 싶은 중고등학생
- 전공과 직업을 연결해보고 싶은 대학생
- 직무 전환 또는 취업 준비를 시작하는 사용자
- 학생의 진로 탐색을 도와야 하는 교사, 상담자, 멘토

## 주요 기능

| 기능 | 설명 |
| --- | --- |
| 회원가입/로그인 | 사용자 계정 생성과 로그인 |
| 진로 검사 | Holland 유형 기반 검사 제출 |
| 검사 결과 저장 | 사용자별 검사 결과와 추천 직업 저장 |
| 직업 정보 | 직업 100종 탐색 및 상세 조회 |
| 멘토 정보 | 직업별 멘토 5명 연결 |
| 멘토링 신청 | 사용자, 멘토, 직업 기준 신청 저장 |
| 관심 직업 | 직업 상세에서 저장하고 마이페이지에서 확인 |
| 채용공고 | 직업과 연결된 채용공고 조회 |
| 커뮤니티 | 공지, 질문, 후기, 새소식 조회 |
| 마이페이지 | 프로필, 로드맵, 검사 결과, 관심 직업, 멘토링 내역 확인 |

## 기술 구조

```text
React + Vite
→ Axios REST API
→ Spring MVC Controller
→ Service
→ Spring Data JPA Repository
→ MySQL
```

## 기술 스택

- Frontend: React, Vite, React Router, Axios
- Backend: Spring Boot, Spring MVC, Spring Data JPA
- Database: MySQL
- Migration: Flyway
- Password: BCrypt
- Architecture: REST API, MVC, Controller-Service-Repository 3계층

## 데이터 구성

- 직업 100개
- 직업별 Holland 유형 3개
- 직업당 멘토 5명
- 총 멘토 500명
- 채용공고 12건
- 커뮤니티 게시글 8건
